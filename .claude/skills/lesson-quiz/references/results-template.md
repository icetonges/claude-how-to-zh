# 结果报告模板

步骤 5（"评分并呈现结果"）所使用的输出格式。填充方括号中的占位符，标题和表格列名保持不变。

```markdown
## Lesson Quiz Results: [课程名称]

**Score: N/10** — [等级标签]
**Quiz timing**: [课前 / 课中 / 课后]
**Question breakdown**: N 道概念题正确，N 道实操题正确

### Per-Question Results

| # | Category | Question (short) | Your Answer | Result |
|---|----------|-----------------|-------------|--------|
| 1 | Conceptual | [题目摘要] | [用户的作答] | [Correct / Incorrect] |
| 2 | Practical | ... | ... | ... |
| ... | ... | ... | ... | ... |

### Incorrect Answers — Review These

[对每道答错的题目，展示：]

**Q[N]: [题目完整文本]**
- Your answer: [用户选择的内容]
- Correct answer: [正确选项]
- Explanation: [为什么这是正确的]
- Review: [课程 README 中应重读的具体章节]

### [按时机定制的提示信息]

[如果是课前测验]：
**Pre-test score: N/10.** 这为你建立了一个基线水平！学习时请重点关注答错的主题。学完这节课后，可以重新测验来衡量进步。

[如果是课中测验]：
**Progress check: N/10.** [如果 7 分以上：进展不错，继续保持！如果 4-6 分：建议先复习答错的主题再继续。如果低于 4 分：建议从头重新阅读。]

[如果是课后测验]：
**Mastery check: N/10.** [如果 9-10 分：你已经掌握了这节课！可以继续下一节。如果 7-8 分：就差一点了——复习答错的主题后重新测验。如果低于 7 分：建议多花些时间学习这节课，尤其是上面标注的章节。]

### Recommended Next Steps

[根据分数与测验时机：]
- [如果已掌握]：按学习路线图进入下一节课：[下一节课链接]
- [如果熟练]：复习这些具体章节，然后重新测验：[章节列表]
- [如果发展中或更低]：重新完整阅读这节课：[课程链接]。重点关注：[薄弱类别列表]
- [提供选项]："你想重新测验，还是换一节课，或者针对某个主题深入讲解？"
```
